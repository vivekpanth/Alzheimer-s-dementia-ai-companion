Memory Companion Agent
An AI-powered web app that helps Alzheimer's patients have warm, personalised conversations — using their own life stories, photos, and memories. Caregivers get daily summaries automatically.
Agentic AI 3 Specialised Agents LangGraph RAG + Memory 10 Weeks Browser-based
The Problem

Over 55 million people live with dementia worldwide. Caregivers spend hours doing reminiscence therapy — talking through old memories to improve mood. There is no affordable AI system that does this automatically.

Our Solution

A caregiver uploads a patient's photos + biography once. The AI then holds daily personalised conversations, detects if the patient is upset, redirects gently, and emails a daily summary to the family.

How it works — 3 agents behind every conversation
Reminiscence Agent
Retrieves personal facts from the patient's uploaded biography and past sessions. Generates warm, grounded responses using only real memories.
Mood Agent
Reads every message for emotional tone — happy, confused, or distressed. If distress is detected, it signals the supervisor to redirect the conversation gently.
Report Agent
At session end, auto-generates a plain-English daily summary for the caregiver — topics discussed, mood trend, and any concerns to follow up on.
